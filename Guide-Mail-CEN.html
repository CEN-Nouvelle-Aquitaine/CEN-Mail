1. Présentation de Mail-CEN

Mail-CEN est une extension Thunderbird développée pour faciliter la migration vers Microsoft 365 au sein du CEN Nouvelle-Aquitaine. Elle permet de :

  • Se connecter à Microsoft 365 pour piloter Outlook depuis Thunderbird
  • Ajouter le compte Outlook dans Thunderbird en quelques clics
  • Migrer les dossiers locaux (mails archivés sur votre ordinateur) vers les archives Outlook
  • Créer les catégories Outlook automatiquement à partir des étiquettes Thunderbird
  • Synchroniser les étiquettes vers les catégories Outlook sur les mails existants
  • Exporter les contacts Thunderbird au format vCard
Compatibilité
Thunderbird 115 et versions ultérieures (y compris la version 140 ESR et au-delà), sur Windows, macOS et Linux.

2. Installation

1
Récupérez le fichier mail-cen-v5.3.xpi fourni par votre service informatique.
2
Dans Thunderbird, cliquez sur le menu (☰) puis Extensions et thèmes.
3
Cliquez sur la roue dentée en haut à droite puis Installer un module depuis un fichier…
4
Sélectionnez le fichier mail-cen-v5.3.xpi puis Ajouter et confirmez l'installation.
5
L'icône Mail-CEN apparaît dans la barre d'outils de Thunderbird. Cliquez dessus pour ouvrir le panneau.
L'icône n'apparaît pas ?
Faites un clic droit sur la barre d'outils → Personnaliser la barre d'outils… → Glissez l'icône Mail-CEN vers la barre → Terminé.

3. Connexion M365

Cet onglet permet de connecter Mail-CEN à votre compte Microsoft 365. Cette connexion est nécessaire pour créer les catégories Outlook et synchroniser les étiquettes.

1
Ouvrez le panneau Mail-CEN et cliquez sur l'onglet Connexion M365.
2
Cliquez sur le bouton « Se connecter avec Microsoft ».
3
Une fenêtre de connexion Microsoft s'ouvre. Connectez-vous avec votre compte CEN (prenom.nom@cen-na.org).
4
Une fois connecté, le statut passe à « Connecté » avec une coche verte. La connexion reste active environ 1h30.
Pourquoi cette connexion ?
Mail-CEN utilise l'API Microsoft Graph pour créer vos catégories Outlook et les appliquer sur vos mails. Sans cette connexion, seules la migration de dossiers et l'export de contacts fonctionnent.

4. Connexion Outlook — Ajouter le compte mail

Cet onglet vous aide à ajouter votre boîte mail Microsoft 365 dans Thunderbird. Si votre compte Outlook est déjà configuré dans Thunderbird, vous pouvez passer cette étape.

1
Dans l'onglet Connexion Outlook, entrez votre adresse email CEN (ex: prenom.nom@cen-na.org).
2
Cliquez sur « Vérifier ». Mail-CEN détecte automatiquement les paramètres de connexion Microsoft 365.
3
Vous pouvez modifier le nom du compte qui apparaîtra dans Thunderbird (par défaut : « Outlook »).
4
Suivez les instructions affichées pour finaliser la configuration dans les paramètres de Thunderbird.
ParamètreValeur

Le compte est déjà configuré ?
Si Mail-CEN affiche « Ce compte est déjà configuré dans Thunderbird », votre boîte Outlook est déjà présente. Passez directement aux étapes suivantes.
La configuration via Mail-CEN ne fonctionne pas ?
Certaines versions de Thunderbird ne permettent pas l'ajout automatique de compte depuis une extension. Dans ce cas, vous pouvez configurer votre compte Outlook manuellement en quelques minutes. Consultez l'Annexe — Ajouter un compte Outlook manuellement en fin de document pour un tutoriel détaillé pas à pas.

5. Migration Dossiers — Migrer les dossiers locaux

Cet onglet permet de copier ou déplacer vos dossiers locaux (mails stockés sur votre ordinateur, dans la section « Dossiers locaux » de Thunderbird) vers les archives de votre compte Outlook. Vos mails seront ainsi accessibles depuis n'importe quel appareil.

Important — Avant de commencer
  • Assurez-vous que votre compte Outlook est bien ajouté dans Thunderbird (étape précédente)
  • La migration peut prendre du temps si vous avez beaucoup de mails — ne fermez pas Thunderbird pendant l'opération
  • En cas d'interruption, vous pouvez relancer sans risque : les doublons sont automatiquement détectés et ignorés
1
Dans l'onglet Migration Dossiers, sélectionnez le dossier source dans « Dossiers locaux ».
2
Sélectionnez le dossier de destination dans votre compte Outlook (par exemple : « Archives »).
3
Choisissez le mode : Copier (conserve les originaux) ou Déplacer (supprime les originaux après transfert).
4
Cliquez sur « Démarrer la migration ». Une barre de progression indique l'avancement.
5
À la fin, un résumé s'affiche : nombre de messages traités, doublons ignorés, et éventuelles erreurs.
Gestion des doublons
Si vous relancez une migration déjà effectuée (par exemple après une interruption), Mail-CEN compare les identifiants uniques de chaque message. Les mails déjà présents dans la destination sont automatiquement ignorés — aucun doublon ne sera créé.

6. Correspondances — Associer étiquettes et catégories

Les étiquettes Thunderbird et les catégories Outlook sont deux systèmes différents pour classer vos mails. Cet onglet fait le lien entre les deux.

Création automatique

1
Assurez-vous d'être connecté à M365 (onglet Connexion M365 → statut vert).
2
Dans l'onglet Correspondances, cliquez sur « Créer automatiquement dans Outlook ».
3
Mail-CEN lit vos étiquettes Thunderbird, crée les catégories correspondantes dans Outlook, et remplit la table de correspondance automatiquement.

Correspondance manuelle

Vous pouvez aussi ajuster le mapping manuellement :

  • Chaque ligne montre une étiquette Thunderbird (à gauche) et le nom de la catégorie Outlook correspondante (à droite)
  • Vous pouvez modifier le nom de la catégorie Outlook dans le champ texte
  • Mettez « — Ignorer — » pour ne pas synchroniser une étiquette
  • Cliquez sur « Sauvegarder le mapping » pour enregistrer vos choix

7. Étiquettes → Catégories — Synchroniser

Une fois la correspondance définie, cet onglet applique les catégories Outlook sur vos mails existants, en se basant sur les étiquettes Thunderbird.

1
Dans l'onglet Étiquettes → Catégories, sélectionnez le compte source (votre boîte avec les étiquettes).
2
Sélectionnez le compte destination (votre compte Outlook/M365).
3
Cliquez sur « Analyser ». Mail-CEN compare les deux boîtes et affiche un résumé par catégorie : combien de mails à synchroniser, combien non trouvés dans la destination.
4
Vérifiez le résumé. Cochez ou décochez les catégories que vous souhaitez appliquer.
5
Cliquez sur « Appliquer via Graph ». Mail-CEN retrouve chaque mail dans Outlook par son identifiant unique, puis ajoute les catégories correspondantes.
Comment ça marche ?

Chaque email possède un identifiant unique (Message-ID). Mail-CEN l'utilise pour retrouver le même mail dans Outlook, puis applique la catégorie via l'API Microsoft Graph. Les étiquettes d'origine dans Thunderbird ne sont pas modifiées.

Étiquettes sans correspondance
Si des étiquettes n'ont pas de correspondance définie, elles apparaissent dans un avertissement. Retournez dans l'onglet Correspondances pour compléter le mapping, puis relancez l'analyse.

8. Export Contacts

Cet onglet permet d'exporter les contacts de votre carnet d'adresses Thunderbird au format vCard (.vcf), compatible avec Outlook et les autres clients de messagerie.

1
Dans l'onglet Export Contacts, sélectionnez le carnet d'adresses à exporter.
2
Cliquez sur « Exporter ». Un fichier .vcf est téléchargé.
3
Pour importer dans Outlook : ouvrez outlook.office.com/people → Gérer les contacts → Importer des contacts → sélectionnez le fichier .vcf.

9. Questions fréquentes

La connexion M365 a expiré, que faire ?

La connexion est valide environ 1h30. Retournez dans l'onglet Connexion M365 et cliquez à nouveau sur « Se connecter avec Microsoft ».

La migration a été interrompue, puis-je la relancer ?

Oui, sans risque. Mail-CEN détecte automatiquement les mails déjà présents dans la destination grâce à leur identifiant unique. Les doublons sont ignorés — seuls les mails manquants sont transférés.

Puis-je utiliser Mail-CEN sur plusieurs postes ?

Oui. L'extension utilise un identifiant fixe qui est le même sur tous les postes. La configuration Azure fonctionne partout sans modification.

Mes étiquettes Thunderbird sont-elles modifiées ?

Non. Mail-CEN lit vos étiquettes mais ne les modifie jamais. Il ajoute uniquement des catégories côté Outlook.

Comment vérifier que les catégories sont appliquées dans Outlook ?

Dans Outlook Web, utilisez la barre de recherche et tapez category:"Nom de la catégorie" pour afficher tous les mails classés dans cette catégorie.

L'icône Mail-CEN n'apparaît pas dans la barre d'outils

Faites un clic droit sur la barre d'outils → Personnaliser la barre d'outils… → cherchez l'icône Mail-CEN et glissez-la vers la barre.

J'obtiens une erreur lors de la migration

Vérifiez que le compte Outlook est bien configuré et que ses dossiers sont synchronisés (cliquez sur le compte dans la barre latérale pour le déplier). Relancez ensuite la migration — les mails déjà transférés ne seront pas dupliqués.
