📅 Export manuel
L'API Thunderbird ne permet pas l'export automatique des calendriers.
Procédure :
1. Dans Thunderbird → menu Événements et tâches → Exporter…
2. Sélectionnez le calendrier → OK
3. Choisissez format iCalendar (.ics) → Enregistrer
4. Dans Outlook → Fichier → Ouvrir et exporter → Importer → Fichier .ics

ℹ️ Pourquoi cette connexion ?
Outlook Online gère ses catégories via l'API Microsoft Graph, indépendamment d'IMAP. Cette connexion permet à Mail-CEN d'appliquer vos catégories directement dans Outlook Online.

Sécurité : aucun mot de passe n'est stocké dans le plugin. La connexion utilise OAuth2 avec votre compte Microsoft CEN.
